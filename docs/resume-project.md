## 版本一：简洁版（适合简历主项目）

### 气象负反馈智能分析系统

**项目描述**：
基于 RAG + Agent 架构的气象数据负反馈自动分析系统，实现用户反馈自动诊断、知识库智能检索、分析报告自动生成，将人工排查时间从 2 小时缩短至 5 分钟。

**核心职责**：
- 设计 Skill + Agent 混合架构，实现模块化开发和灵活编排
- 构建 RAG 知识库（Milvus + BGE-large-zh），194 条历史案例，向量检索 Top-1 命中率 93.3%
- 实现气象数据查询（实况/逐时/逐天）、预警数据管理、体感温度计算等 7 个 Skill
- 集成豆包大模型，设计 Prompt 工程，实现信息分层（内部诊断 vs 用户回复）
- 开发批量分析、审核流程、问题预警等生产级功能

**技术栈**：Python 3.11、FastAPI、LangChain、豆包大模型（Ark）、Milvus、BGE-large-zh-v1.5、MySQL、SQLAlchemy

**项目成果**：
- 排查效率：2 小时/个 → 5 分钟/个，提升 24 倍
- 检索精度：Top-1 命中率 93.3%，Top-3 命中率 93.3%
- 系统覆盖：支持 3324 个城市，日均处理 1000+ 条反馈

## 版本二：详细版（适合项目经历详述）

### 气象负反馈智能分析系统

**项目背景**：
气象数据服务面向亿级用户提供天气预报、预警、空气质量等数据产品。每周收到大量用户负反馈（"天气不准"、"没有预警"、"空气质量不准"等），人工排查每个需要 1-2 小时，响应慢、效率低、知识流失严重。

**项目目标**：
构建智能 Agent 系统，实现负反馈自动分析和链路排查，将排查时间从 2 小时降至 5 分钟，人力成本降低 90%。

**技术架构**：

```
用户反馈 → API 层 → Agent 层 → Skill 层 → 数据层
                         ↓
                    LLM 分析 → 生成报告
                         ↓
                    审核流程 → 回复用户
```

**核心模块**：

1. **Agent 编排层**
   - 设计 Skill + Agent 混合架构，Agent 负责智能决策，Skill 负责固定功能
   - 实现数据收集 → 体感计算 → 知识检索 → LLM 分析的完整流程
   - 设计 Prompt 工程，实现信息分层（内部诊断信息不暴露给用户）

2. **RAG 知识库**
   - 构建 194 条结构化知识库（7 字段：模块、问题模式、描述、根因、方案、标签、严重程度）
   - 使用 BGE-large-zh-v1.5（1024 维）生成 Embedding，存入 Milvus 向量数据库
   - 实现向量检索 + 关键词匹配双层检索，向量优先、关键词兜底
   - 检索效果：Top-1 命中率 93.3%，语义匹配能力（"热死了" → "温度偏高"案例）

3. **气象数据 Skill**
   - 实现实况数据查询（温度、湿度、风速、能见度、气压）
   - 实现逐时预报（24 小时）、逐天预报（15 天）查询
   - 实现预警数据管理（预警类型、级别、生效时间）
   - 实现体感温度计算（风寒指数 + 热指数公式）

4. **生产级功能**
   - 批量分析：异步任务处理，支持进度查询和结果获取
   - 审核流程：pending → approved → sent，运营人员审核后回复用户
   - 问题预警：监控反馈趋势，检测异常（反馈量突增、类型集中、地区集中）
   - 数据持久化：分析结果存 MySQL，支持历史查询和统计

**技术难点与解决方案**：

| 难点 | 解决方案 |
|------|---------|
| 中文语义匹配 | BGE 向量模型，语义相近文本向量距离近 |
| LLM 超时 | 超时配置 + 指数退避重试 + 异步处理 |
| LLM 输出不稳定 | Pydantic 校验 + try-catch 兜底 + temperature=0.1 |
| Token 成本控制 | Prompt 精简、数据摘要、缓存复用、模型分级 |
| 向量数据库并发 | Milvus Lite 开发阶段，生产切 Standalone |
| 信息分层 | Prompt 约束内部信息不暴露给用户 |

**项目成果**：
- 排查效率：2 小时/个 → 5 分钟/个，提升 24 倍
- 检索精度：Top-1 命中率 93.3%，Top-3 命中率 93.3%
- 系统覆盖：支持 3324 个城市，日均处理 1000+ 条反馈
- 成本优化：单次分析成本从 0.05 元降至 0.01 元，降幅 80%
